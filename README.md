# DailyFruit Backend v4

Node.js + Express + MongoDB backend for Daily Fruit Co.

Features:

- User signup, password login, email OTP login
- JWT auth (user + admin roles)
- Plans API for frontend index/plans pages
- Razorpay payments + verification
- Subscriptions auto-created after successful payments
- Profile API for dashboard
- Admin APIs for login, stats, users, payments
- Deliveries API (for future UI)
